# Проект: Асинхронный парсер вакансий из Telegram-каналов

**Октябрь 2024 | Фриланс**

- Разработал парсер на базе Telethon с поддержкой авторизации, хранения сессии и 2FA.
- Реализовал неблокирующий сбор сообщений с использованием asyncio.
- Настроил обработку ограничений Telegram API с динамическими задержками для соблюдения лимитов запросов.
- Реализовал многоуровневую фильтрацию сообщений по дате, ключевым словам, длине и наличию текста.
- Добавил нормализацию временных зон (UTC) для корректной фильтрации.
- Вынес конфигурацию каналов и ключевых слов во внешние файлы (config.ini, Excel-файл).
- Реализовал экспорт отобранных сообщений в Excel (дата, текст, ссылка, автор) при помощи pandas.

### Стек
`Python`, `asyncio`, `Telethon`, `pandas`, `configparser`
